2/9 - 5/9

1. 카테고리 계층화
2. 데이터 캐싱

1. 분류없음 카테고리를 기본적으로 추가해서 부모 카테고리가 없는 키워드들을 모아놓는다
2. 정정버튼은 따로 없이 표의 카테고리 태그 부분을 클릭하면 인풋박스와 확인버튼이 나오게 한다.
3. 네비게이션바에 등록버튼을 추가해서 그 버튼을 누르면 부모-자식 카테고리를 통째로 등록할 수 있는 모달이 나온다
4. 상위 카테고리를 클릭하여 나오는 등록 버튼을 눌렀을때는 하위 카테고리를 2-3개까지 추가할 수 있도록 한다.
5. 하위 카테고리를 클릭하여 나오는 등록 버튼을 눌렀을때는 키워드를 5개까지 추가할 수 있도록한다.
